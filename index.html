Overview

This is the home page for my homework 4 implementation. For this project, I was able to fully implement a node link visualization with a Cluster, and a space-filling visualization using a Treemap. The goal when creating these visualizations was to see the different types of fire department calls that occurred of April 20th. Because of San Francisco's unique ties to April 20th (4/20), I expected to see calls due excess smoke, or potentially some unusual medical calls. In addition to this, I was curious to see if I could tell which neighborhoods (such as maybe Haight Ashbury) participated the most. While Each of the visualizations that I created have their different pros and cons, I believe that they provide us with some interesting insights into how San Francisco celebrated 4/20.

Data

This data set is composed of information over service calls to San Francisco's Fire Department, which was made available by the city here. Using their public website, I was able to filter by the relevant date, and then roll up the data into a few columns, with approximately 120 rows of data. The relevant columns that I used in these visualizations are:

  • Neighborhooods The neighborhood that is relevant to call.
  • Call Type The category of this report.
  • Call Number: This column is a unique identifier that was used in the rollup to generate a count for each emergency, by neighborhood.
