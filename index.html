Overview

This is the home page for my homework 3 implementation. For this project, the base requirements for an was to implement a some form of map in D3, and to then add two levels of interactivity on top of that. To meet this requirement, I was able to fully implement a Choropleth Map, as well as one level of interactivity (details on demand). My inspiration for this project was to build a visualization that I thought was unique, and would clearly convey the information that I wanted to share. I believe I was able to achieve this goal.

Data

This data set is composed of information over 4 million reports to San Francisco's 311 hotline, which was made available by the city here. Using their public API, I was able to filter by a couple relevant columns, to get the exact data I wanted. The relevant columns that I used in this visualization are:

  • Opened: The Date and time this report was opened. I used this column to filter down to only cases that were opened in February.
  • Category: The category of this report. I filtered down the category to focus on just abandoned vehicles.
  • Request Details: This column contains additional information about the request. I used it to filter out reports with "DPT Abandoned Vehicles Low". This is a special case designate by the city, which I deemed irrelevant to this visualization.
  • Neighborhood: This is the neighborhood that the report is from. In D3, I used this column to generate a count of abandoned vehicles for each neighborhood.
